kdmccormick plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/kdmccormick/tutor-contrib-kdmccormick

Usage
-----

::

    tutor plugins enable kdmccormick
